# Como usar no G HUB:

## Abra o Logitech G HUB.
1. Selecione o perfil do jogo (ou crie um novo para o jogo desejado).

2. Vá em Scripting (ou "Scripts").

3. Crie um novo script Lua.

5. Cole O Codigo.

4. Clique em Salvar e Executar (ou Ctrl + S).

5. Ative o CapsLock (ou a tecla que você escolheu) para ligar o recoil control.

6. No jogo: Segure o botão direito do mouse (mirar) + botão esquerdo (atirar).

## Dicas de personalização:

1. Aumentar/diminuir o recuo → mude o RecoilMode para "Low", "High", "Ultra" ou coloque "Custom" e ajuste o CustomStrength.

2. Quer compensação horizontal? Mude HorizontalStrength = 0 para 2 ou -2 (positivo = direita, negativo = esquerda).

3. Quer sempre ativo (sem precisar de CapsLock)? Mude RequireToggle = false.

4. Para jogos específicos (ex: Rainbow Six, CS2, Valorant), teste os valores de força entre 4 e 12.
