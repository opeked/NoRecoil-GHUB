# Como usar no G HUB:

## Abra o Logitech G HUB.
Selecione o perfil do jogo (ou crie um novo para o jogo desejado).
Vá em Scripting (ou "Scripts").
Crie um novo script Lua.
Apague tudo que estiver lá e cole o código acima.
Clique em Salvar e Executar (ou Ctrl + S).
Ative o CapsLock (ou a tecla que você escolheu) para ligar o recoil control.
No jogo: Segure o botão direito do mouse (mirar) + botão esquerdo (atirar).

## Dicas de personalização:

Aumentar/diminuir o recuo → mude o RecoilMode para "Low", "High", "Ultra" ou coloque "Custom" e ajuste o CustomStrength.
Quer compensação horizontal? Mude HorizontalStrength = 0 para 2 ou -2 (positivo = direita, negativo = esquerda).
Quer sempre ativo (sem precisar de CapsLock)? Mude RequireToggle = false.
Para jogos específicos (ex: Rainbow Six, CS2, Valorant), teste os valores de força entre 4 e 12.
